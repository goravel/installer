Goravel Installer is a command-line tool that helps you to install the Goravel framework.

## Version

| goravel/installer | goravel/framework |
|-------------------|-------------------|
| v1.1.x            | v1.13.x           |

## Installation

```bash
go install -u github.com/goravel/installer
```

## Usage

```bash
goravel new blog
```
